# Portfolio V4 - Cat Hoang

A modern portfolio website built with React and Vite, implementing the design from Figma.

## Features

- **Hero Section**: Retro computer illustration with interactive popup window
- **Project Grid**: 12-card grid showcasing various design projects
- **About Section**: Three-card layout featuring personal information, design values donut chart, and interests mind map

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18

## Design

This portfolio implements the "Home_Folded Paper, 2.5D" design from Figma, featuring:
- Clean, minimalist aesthetic
- Light gray color scheme with subtle patterns
- Responsive grid layouts
- Interactive elements with hover effects
